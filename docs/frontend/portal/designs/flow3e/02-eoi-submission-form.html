Single input — message / cover note (required). Decision: a dedicated page (not a modal). Rationale below. Post-submit confirmation is shown inline as a banner on return to the CFEOI detail (Block B), matching flow3c/02.

DECISION Page over modal. The EOI route is linkable and the cover note deserves a calm, full-width writing space; a page keeps parity with the app's other create surfaces (proposal / CFEOI editors) and makes Cancel → back-to-CFEOI unambiguous. A modal on the detail page would also be acceptable for such a small form.
